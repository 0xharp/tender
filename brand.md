## Tone and voice

### Words to use

Direct, specific, number-forward, factual. Verbs over adjectives, nouns over metaphors. Lean into restraint — "considered" not "amazing", "sealed" not "ultra-private". Premium words land harder when used sparingly. Describe what something does, not what it feels like.

### Words to avoid

Hype words ("revolutionary", "game-changing", "unleash", "supercharge", "powerful", "seamless"), exclamation marks, emojis, "simply" / "just" / "easy", "luxury" / "exclusive" / "elite" (using them makes it less premium), and stacked adjectives. "Unlock" is banned. No urgency ("limited time", "act now") unless the situation genuinely is.

### Voice example

> "Sealed bid committed. Buyer can decrypt at the reveal window. Withdraw any time before close — full rent refunded."

## Usage dos and don'ts

**Do:**

- Use shadcn CSS variables (`bg-background`, `text-foreground`, `bg-primary`, `bg-card`, `border-border`) everywhere. Never hardcode hex.
- Use `font-mono tabular-nums` for any number that updates in place (USDC amounts, bid counts, time-remaining).
- Use `font-mono` for hashes, PDAs, tx signatures (already the convention in the codebase).
- Reach for the gradients on **hero / landing / decryption-reveal** surfaces only — those are the "selective glass + chromatic moments" the brand direction reserves.
- Test every component in both light and dark mode before considering it done.
- Respect the 4 px spacing grid (`gap-2`, `p-4`, `px-6`).

**Don't:**

- Hardcode colors (`#FF0000`, `rgb(...)`) in component files.
- Use `transition: all` — always specify `transition-colors`, `transition-transform`, etc.
- Mix more than one gradient on the same screen.
- Use the accent gradient as a background under body text or dense data tables — gradients fight legibility and pull eyes off content.
- Apply chromatic glass / blur to working surfaces (RFP list, bid composer, dashboards). Those stay calm — Linear bones, not Phantom bones. Glass moments are reserved for hero / landing / reveal surfaces.
- Override shadcn tokens per-component — if a color is wrong, fix it in `globals.css` once.

## Direction recap (the D + B blend)

The Tender brand sits at the intersection of two design references:

- **D — Quiet B2B SaaS bones** (Linear, Resend, Height): Working surfaces — RFP list, bid composer, provider profile, dashboard — stay calm, dense, restrained. Geist body type, muted backgrounds, sharp focus rings, subtle micro-motion. This is where 80% of user time happens.
- **B — Selective glass + chromatic moments** (Phantom new, MagicEden new): Hero / landing / reveal surfaces use the violet, the gradients, the Framer Motion expressive→cinematic tier, backdrop-blur glass. This is what investors and first-time visitors see first.

Animation is **Framer Motion**, expressive→cinematic tier. The hero state to lavish: **decryption reveal** — sealed bid ciphertext "unlocks" into plaintext fields with a violet glow.
