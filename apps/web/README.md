# @tender/web

The Tender frontend — Next.js 16 + Tailwind 4 + (shadcn/ui to be added). All buyer and provider flows live here.

From the monorepo root:

```bash
pnpm install
pnpm --filter @tender/web dev
```

Then open <http://localhost:3000>.

See the root [`README.md`](../../README.md) for the full architecture, and [`AGENTS.md`](./AGENTS.md) for Next.js 16-specific notes.
